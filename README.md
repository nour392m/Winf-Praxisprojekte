# Portfolio: Praxisprojekte Wirtschaftsinformatik

Herzlich willkommen auf meinem GitHub-Profil! Hier sammle ich kleinere, praxisnahe Programmierprojekte, die ich im Rahmen meines Wirtschaftsinformatik-Studiums umsetze, um mein theoretisches Wissen in die Praxis zu überführen.

## 🛠️ Enthaltene Projekte

### 1. Automated Sales Data Analysis Pipeline (`excel_reporter.py`)
* **Technologien:** Python, Pandas, Matplotlib
* **Funktion:** Simuliert das Einlesen von Unternehmensdaten, filtert diese nach Umsatzvorgaben und generiert automatisch einen visuellen Report (Balkendiagramm) im Ausgabeordner.

### 2. SQLite Database Integration (`database_project.py`)
* **Technologien:** Python, SQL, SQLite
* **Funktion:** Erstellt eine relationale lokale Kundendatenbank, befüllt diese automatisiert mit Testdatensätzen und führt komplexe SQL-Abfragen (`WHERE`-Filterungen) durch.
